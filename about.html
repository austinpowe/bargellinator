Bargellinator is a generative* bargello* pattern designer.

*Generative here means that we're giving computers a small set of constrained inputs and seeing what it kicks out at the end!

*Bargello is an embroidery technique that usees yarn on canvas mesh.

Each time you reload the page, you get a never-before-seen bargello tapestry.

Bargello is an embroidery technique that became popular in the mid-70s but fell into unpopularity in the succeeding decades. It uses brightly colored yarn on canvas or plastic mesh to create repeating patterns.

Because of bargello's waning popularity, there simply aren't as many patterns available to the modern stitcher as there are for other fiber arts, like knitting, cross-stitch, embroidery, or weaving.

This site my personal attempt to decode what makes bargello patterns so fun and unique, while hopefully providing inspiration to other stitchers. If you're interested in how it works, you can peek at the raw (but annotated!) Javascript code here.
